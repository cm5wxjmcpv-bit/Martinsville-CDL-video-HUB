- /assets/martinsville-cdl-bg.png      (background image)
- /assets/martinsville-cdl-logo.png    (header logo)

How to publish (free):
1) Create a new public repo on GitHub (e.g., Martinsville-CDL-video-HUB)
2) Upload the entire folder contents to the repo root (keep the /assets folder).
3) In the repo: Settings → Pages → Build and deployment → Deploy from a branch → main / root → Save
4) Open the Pages URL shown by GitHub. Hard refresh if you update images later (Cmd/Ctrl+Shift+R).
